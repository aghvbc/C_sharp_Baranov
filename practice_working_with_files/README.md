## 1. Рекурсивный подсчет файлов и папок

### ***Задание:***
Напишите программу, которая рекурсивно проходит по указанной папке и считает:
- Количество всех файлов
- Количество всех подкаталогов
- Общий размер файлов в папке и всех подкаталогах
- Дополнительно: выводить список файлов больше 10 МБ.

### ***Используемые технологии:***

| Технология   | Версия  |
|--------------|---------|
| C#           | 13      |
| .NET SDK     | 9.0.110 |
| .NET Runtime | 9.0.9   |

### ***Используемые библиотеки:***
- System
- System.IO

### ***Инструкция по запуску:***
1. Полностью замением весь Program.cs
2. Проверяем работоспособность


## 2. Фильтрация и копирование файлов

### ***Задание:***
Напишите программу, которая копирует все файлы с расширением .txt и .json из папки Source в папку Backup.
При копировании добавлять к имени файла текущую дату
Пропускать файлы размером больше 5 МБ

### ***Используемые технологии:***
| Технология   | Версия  |
|--------------|---------|
| C#           | 13      |
| .NET SDK     | 9.0.110 |
| .NET Runtime | 9.0.9   |

### ***Используемые библиотеки:***
- System
- System.IO

### ***Инструкция по запуску:***
1. Полностью замением весь Program.cs  
2. Запускаем
3. Смотрим где нужно создать .txt/.json файлы
4. Создаём файлы
5. Проверяем работоспособность
